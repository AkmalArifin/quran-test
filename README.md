### Quran Test

This is for me to try display quran using v1 font

#### How to Run:
- Run `npm install`
- Create .env file in `api` folder. Fill it with your MySql account
```
DB_USER=
DB_PASSWORD=
```
- Create database name `srq_project` and import the db file to the database in MySql. The file path is `./api/db/db.sql`
- Run `index.html` using Live Server
